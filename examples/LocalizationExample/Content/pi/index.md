Haul up a barrel an' take a seat, matey! This here establishment be servin' documentation in four fine tongues, arrr!

## What We Be Servin'

We got ourselves a treasure map o' knowledge, lovingly translated into:

- **English** -- the common tongue o' landlubbers
- **Pig Latin** -- fer the classically trained swabbies
- **Swedish Chef** -- fer those who prefer their docs with a side o' bork
- **Pirate** -- the one true language o' the seven seas! Arrr!

## How It Works, Ye Bilge Rat

This here site uses **folder-based locale routin'**. The default tongue (English) be livin' at the root, while translations be stowed in subfolders like treasure in a chest.

Try switchin' languages usin' the language picker in the top-right corner o' the header. Yo ho ho!
