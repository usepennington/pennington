Pull up a chair, friend! This humble establishment serves documentation in four fine languages.

## What We Serve

We have a menu of knowledge, lovingly translated into:

- **English** -- the house common tongue
- **Pig Latin** -- for the classically trained
- **Swedish Chef** -- for those who prefer their docs with a side of bork
- **Pirate** -- for the seafaring folk

## How It Works

This site uses **folder-based locale routing**. The default language (English) lives at the root, while translations live in subfolders:

```
Content/
  index.md              <- you are here (English)
  pl/index.md           <- Pig Latin
  sv/index.md           <- Swedish Chef
  pi/index.md           <- Pirate
```

Try switching languages using the language picker in the top-right corner of the header.
