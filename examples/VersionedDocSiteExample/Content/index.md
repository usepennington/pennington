This docsite publishes two parallel documentation trees from a single host — one per major version of `Humanizer.Core`. The two version trees are independent: each has its own content area, its own API reference, and its own sidebar.

- [v1 (2.8.26)](/v1/) — staged via `<PackageDownload>` so it coexists with the active reference.
- [v2 (2.14.1)](/v2/) — resolved via `FromPackageReference("Humanizer")`.

The sidebar area selector doubles as a version switcher.
