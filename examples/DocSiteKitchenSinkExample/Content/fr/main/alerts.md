# Alertes et encadrés

Pennington reconnaît la syntaxe d'alerte inspirée de GitHub : un
blockquote dont la première ligne est `[!KIND]` devient une alerte
stylisée.

> [!NOTE]
> Les notes apportent une information latérale que le lecteur doit
> garder en tête avant de continuer.

> [!TIP]
> Les astuces pointent vers un raccourci, un réglage par défaut
> intelligent ou un motif qui garde le cas courant simple.

> [!IMPORTANT]
> Les encadrés importants mettent en évidence un contenu structurant.

> [!WARNING]
> Les avertissements préviennent d'un résultat incorrect que le lecteur
> peut encore éviter.

> [!CAUTION]
> Les mises en garde signalent quelque chose d'irréversible — options de
> CLI destructrices, changements de format cassants, failles de
> sécurité.
