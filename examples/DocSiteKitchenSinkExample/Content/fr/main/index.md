# Documentation exhaustive

Bienvenue dans l'exemple **DocSite Kitchen Sink**. Ce site câble
volontairement chaque surface configurable de DocSite afin que
chaque recette how-to pointe vers un fichier ou un symbole `xmldocid`
concret.

## Ce que vous trouverez

- Deux zones de contenu — **Main** et **API** — chacune avec sa propre
  page d'accueil.
- Deux locales — anglais à la racine, français sous `/fr/`.
- Un composant Mdazor personnalisé — `<FeatureCallout>` — présenté sur
  la page [composants UI](/main/ui-components-in-markdown).
