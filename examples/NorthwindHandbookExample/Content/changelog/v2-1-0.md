### Breaking: Auth token format changed

The authentication token format has been updated to use JWT v2. All existing tokens will need to be refreshed.

```csharp
// Old format
var token = AuthService.GenerateToken(user);

// New format - now returns a structured result
var result = AuthService.GenerateTokenAsync(user);
var token = result.Token;
var expiry = result.ExpiresAt;
```

#### Migration Steps

1. Update all token generation calls to use the async API
2. Handle the new `TokenResult` return type
3. Deploy token refresh endpoint before switching
