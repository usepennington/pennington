Reference for `MonorailCssOptions` and the color scheme system. Document the `IColorScheme` interface. Document `NamedColorScheme` with its 5 properties (PrimaryColorName, AccentColorName, TertiaryOneColorName, TertiaryTwoColorName, BaseColorName) and valid Tailwind color names. Document `AlgorithmicColorScheme` with PrimaryHue (0-360), BaseColorName, and ColorSchemeGenerator. Explain the 5 semantic color roles and how they map to CSS custom properties. Document `CustomCssFrameworkSettings`, `ExtraStyles`, and `ContentPaths`. List the built-in component styles (code blocks, tabs, alerts, syntax highlighting, prose, search modal) and their CSS class names so users know what they can override. Document `AddMonorailCss` and `UseMonorailCss` extension methods.
