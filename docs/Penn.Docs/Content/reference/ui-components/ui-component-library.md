Complete parameter reference for every Penn.UI Razor component. Use a consistent format per component: component name, namespace, parameters table (name, type, default, description), and usage notes. Components to document: `Badge` (Variant, Size, ChildContent), `Card` (Icon, ChildContent, CSS classes), `CardGrid` (ChildContent), `LinkCard` (Href, Title, Description), `BigTable` (ChildContent), `CodeBlock` (Language [required], Code, ChildContent, IsInTabGroup), `Steps` (Type, ChildContent), `Step` (StepNumber, ChildContent), `TableOfContentsNavigation` (TableOfContents, Section, plus all CSS class parameters), `OutlineNavigation` (ContentSelector, Title, plus all CSS class parameters), `LanguageSwitcher` (AlternateLanguages), `StructuredData` (Article, Breadcrumbs, WebSite), `FallbackNotice`. For `CodeBlock`, note that `Language` is required and how `ChildContent` text extraction with indent normalization works.
