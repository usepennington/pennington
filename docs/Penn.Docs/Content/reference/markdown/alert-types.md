Short reference page documenting the 5 alert types. For each: the markdown syntax (`> [!NOTE]`, etc.), the HTML output structure (`<div class="markdown-alert markdown-alert-note">`), the CSS class name, and the default color mapping. Note GitHub-flavored markdown compatibility — these use the same syntax as GitHub's alert extension. Document the custom alert inline parser that replaces Markdig's built-in one and any behavioral differences.
