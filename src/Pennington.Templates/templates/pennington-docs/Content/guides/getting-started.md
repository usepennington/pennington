# Getting started

This is a sample Guides page so the sidebar has something to render on first
run. Replace it with your real content.

## Adding a page

Drop any `.md` file into `Content/guides/` and save. The dev server picks it up
and adds it to the sidebar automatically — no restart needed.

Front matter controls the page metadata. `title` is required; `description`,
`sectionLabel`, and `order` shape how the page appears in navigation.
